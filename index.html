Build Faster with Arc IDE

Arc IDE is a modern desktop environment built with Qt + Monaco. You get a polished dark interface, extension support, rich themes, and a fast native workflow for daily coding.

DownloadRead Docs

Why Arc IDE

Arc IDE combines web-based UI flexibility with native desktop performance. The frontend is fully customizable with HTML/CSS/JS, while C++ backend APIs handle filesystem and system integration through a clean bridge layer. This architecture gives you a smooth user experience and room to grow.

Modern Editor

Monaco-based editing with language support, tabs, explorer, and custom themes.

  • Font size and theme customization
  • Fast file switching and tabs
  • Syntax support including custom Sway

Extensions

Drop extension folders and extend Arc IDE with custom runtime features.

  • Manifest-based loading
  • Runtime activation API
  • Optional custom extension styles

Native App

Built with Qt WebEngine and WebChannel for performance and desktop integration.

  • Direct filesystem access
  • Desktop dialogs and actions
  • Stable cross-platform architecture
