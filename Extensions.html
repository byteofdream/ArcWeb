Extensions

Create custom tools and UI upgrades by adding extension folders with runtime scripts and optional styles.

Extension Format

Each extension contains a manifest.json file for metadata and a main.js entry script. On startup, Arc IDE discovers installed folders, loads manifests, and activates runtime modules with a lightweight API object.

Simple Structure

Minimal setup keeps extension authoring fast and easy to maintain.

Activation Flow

Load, validate, activate. Failing extensions are isolated without crashing the app.

Capabilities

Access Monaco, editor model state, bridge methods, and UI status actions.
