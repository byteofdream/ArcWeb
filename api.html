Bridge API

The Arc bridge exposes native operations from C++ to JavaScript through QWebChannel with a small and clear API surface.

Design Principles

The API is intentionally compact: simple file access, directory listing, and write support. This keeps the app predictable and secure while making extension development straightforward. All methods are available to the frontend runtime and can be composed into richer workflows.

readFile(path)

Reads local text files and returns UTF-8 content for editor models.

writeFile(path, content)

Persists model changes to disk with truncation and overwrite semantics.

listFiles(path)

Returns normalized tree data for the explorer and folder navigation.
