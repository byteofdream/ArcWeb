Frequently Asked Questions

Answers to common setup questions, extension behavior, and platform expectations.

Is Arc IDE free?

Yes, this project is free for development and experimentation.

How do I change themes?

Open Settings in Arc IDE and pick UI + Monaco theme combinations.

Can I add custom language support?

Yes. Register a Monaco language directly or through an extension package.

Where are extensions stored?

By default in Arc app data directory under the extensions folder.

Will broken extensions crash the app?

No, extension loading failures are isolated and reported without stopping the IDE.

Does Arc IDE require internet?

Core editing works locally. Internet may be needed for CDN-hosted Monaco assets.
